Algorithmic Line Drawing

Select a mode to get started. Each mode uses different paramaters to generate random line art, connecting invisible points on a grid.

In Grid there are 10,078,751,602,022,313,874,633,200 possible line combinatnions that can be shown. In One, you'd have to stay on the website generating drawings for 79,898,779,189,040,400 (79 qaudrillion years) before you would see the same drawing twice. The universe itself is only 13,800,000,000 (13.8 billion) years old. In Chaos and Qaud each drawing is more rare by many orders of magnitude.

Each drawing is likley something that will only ever be seen once.

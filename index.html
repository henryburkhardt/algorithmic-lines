Algorithmic Line Drawing

Select a mode to get started. Each mode uses different parameters to generate random line art, connecting invisible points on a grid.

In Grid there are 10,078,751,602,022,313,874,633,200 possible line combinations that can be shown. In One, you'd have to stay on the website generating drawings for 79,898,779,189,040,400 (79 quadrillion years) before you would see the same drawing twice. The universe is only 13.8 billion years old. In Chaos and Quad each drawing is more rare by many orders of magnitude.

Each drawing is likely something that will only ever be seen once.
